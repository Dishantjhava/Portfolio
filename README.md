## Overview

A fully custom, single-file portfolio website for **Dishant Jhava** — B.Tech CSE student at Lovely Professional University (CGPA 7.6). Built with a dark chemistry-lab aesthetic inspired by Breaking Bad, featuring a cinematic intro animation, particle canvas effects, glitch typography, a periodic-table–style skills display, and a thank-you page after form submission.

---

## Features

- **Cinematic Intro Overlay** — Animated beaker SVG with liquid fill, particle explosion, formula bursts, and a Vimeo background. Auto-dismisses (5s mobile / 12s desktop) or on skip.
- **Custom Cursor** — Dot + lagging ring cursor with hover scale effects (desktop only, disabled on touch devices).
- **Hero Canvas** — Floating particle field rendered on HTML5 Canvas.
- **Glitch Typography** — CSS-only glitch animation on the hero name.
- **Periodic Table Skills** — 20 skills rendered as styled chemical element cards.
- **Scroll Reveal** — IntersectionObserver fade-in + slide-up with fallback timeouts.
- **Animated Stats Counter** — Numbers count up with easing on scroll into view.
- **Typewriter Effect** — Hero subtitle types out with a blinking cursor.
- **Scroll Progress Bar** — Thin green bar tracks reading progress.
- **Active Nav Highlighting** — Nav links update based on current viewport section.
- **Mobile Hamburger Menu** — Full navigation available on all screen sizes.
- **Contact Form + Thank You Page** — Web3Forms powered, redirects to custom `thankyou.html`.
- **Resume Download** — Updated PDF linked in nav and dedicated download section.
- **Scanline FX** — Subtle CRT overlay for atmosphere.
